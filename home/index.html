Connecting to Xinutec

Please use the round robin DNS on irc.xinutec.org. IPv6 connectivity is currently only offered by irc.barfooze.de. Use Port 6667 for normal connections, and Port 7776 for SSL. Web-based access to the network is available here via IPv4.

Channels

#linux is the main channel. If you run a channel, you are free to host a channel homepage (static HTML only) on this server.
